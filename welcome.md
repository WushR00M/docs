# Sunrise Documentation
The official Sunrise documentation wiki for WushR00M Studios' indie game; Sunrise. What's included on this wiki is documentation for various features of the game, including:

1. The ability to customize and modify Sunrise using Lua scripts. You can view the [Getting Started](lua?id=getting-started) section of the Lua Modding page for more information.
2. Upload content to Sunrise's Steam Workshop pages, [seen here](workshop?id=upload).
3. Access content subscribed to via Sunrise's Steam Workshop pages, [also seen here](workshop?id=play).

...and more.
